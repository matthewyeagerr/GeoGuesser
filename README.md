
# Guess the Country by Capital - Python Quiz Game

## Description

A fun and interactive command-line quiz game built in Python that tests your knowledge of world capitals. The game loads country-capital pairs from a text file and quizzes you on 50 random capitals. You can type the country name or type "?" to receive a helpful hint about the capital.

## Features

* Loads country-capital data from a text file with headers "Country" and "Capital"
* Asks 50 random capital questions per game session
* Provides hints on request ("?") showing the first letter and length of the capital city
* Tracks your score and current streak of correct answers
* Displays final accuracy percentage at the end of the game

## How to Play

1. Run the Python script.
2. You will be prompted with a capital city.
3. Type the name of the country that corresponds to the capital.
4. If you need a hint, type "?" to get the first letter and number of letters of the capital.
5. Keep guessing until you answer or want to move on.
6. Your final score and accuracy will be displayed at the end.

## Requirements

* Python 3.x
* A data file named "Capitals.txt" containing the country and capital data with headers "Country" and "Capital"

## How to Use Your Own Data

Replace the "Capitals.txt" file with your own text file formatted with the columns "Country" and "Capital" in the first row, followed by the data rows. Make sure the file is in the same directory as the script.
